• The img element is used to add images to a site
    • src: Relative path to the image
    • alt: Description of the image for screen readers
    • title: Brief hoverover description of the image
    • height: Image height in pixels - Use CSS instead
    • width: Image width in pixels - Use CSS instead
  • The image's position on the page depends on the elements it lives by on the page
    • Images are inline elements but block elements, such as a paragraph, cause an image to appear on a new line
  • Align can be used to align an image relative to text either horizontally or vertically
  • Images should always use the right format and dimensions/resolution
  • The figure and figcaption elements can be used to group an image with a caption
