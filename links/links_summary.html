Links use the a element

They can link to external pages, internal pages, a part of the same page, a part of a different page, a new email, and more

The type of URL used to link depends on whats being linked

External links should use absolute/fullpath URLs and internal links should relative URLs using typical Unix filepath structure

The id element is a way to refer to an element from another element

This way you can link to a specific part of a page

../index.html
